# Airport Management System
## CSPC52 Mini Project

ReactJS + NodeJS and MySQL

## setup

### db
`cd src/db` then `node seed.js`

### backend

1. `npm i` in  root directory
2. Create `src/env/.env` from `src/env/.env.example`
3. Run `node index.js`

### frontend

`cd frontend` then `npm i` then `npm start`
